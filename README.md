# 🎮 Number Guessing Game (GUI Based)

This is a fun and interactive Number Guessing Game built using Python and Tkinter. The game generates a random number between 1 and 100, and the player has to guess the correct number with the help of hints.

The graphical user interface (GUI) makes the game user-friendly and visually engaging. Players can enter their guesses through an input box and receive instant feedback such as whether the number is too high or too low.

## 🚀 Features

* Random number generation for each game session
* Interactive GUI built with Tkinter
* Real-time hints ("Too High" / "Too Low")
* Attempt counter to track performance
* Simple and beginner-friendly design

## 🛠️ Technologies Used

* Python
* Tkinter (for GUI)
* Random module

## 🎯 How to Play

1. Run the program
2. Enter a number between 1 and 100
3. Click the "Check" button
4. Follow the hints and try to guess the correct number
5. Win the game when you guess the correct number

## 💡 Learning Outcomes

* Understanding of GUI development in Python
* Event-driven programming concepts
* Working with loops, conditions, and functions
* Integrating logic with user interface

This project is ideal for beginners who want to practice Python fundamentals while building a fun and interactive application.
